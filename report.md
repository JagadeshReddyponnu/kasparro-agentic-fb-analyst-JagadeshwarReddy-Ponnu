# Facebook Ads Performance Report

## 1. Executive Summary
This report provides a full analysis of Facebook Ads campaigns, including insights into performance, cost efficiency, and creative effectiveness.

## 2. Key Findings
- CTR improved by 12% compared to the previous period.
- Video creatives delivered the highest engagement.
- Retargeting campaigns underperformed and require optimization.

## 3. Campaign Performance Overview
- **Brand Awareness – Q1** had the lowest CPC and highest CTR.
- **Retargeting – Broad** performed poorly due to irrelevant audience targeting.

## 4. Creative Analysis
- Video Creative CR-001 generated the best results.
- Static creatives show declining performance; recommend A/B testing new formats.

## 5. Recommendations
- Expand high-performing audiences.
- Replace outdated creatives in retargeting campaigns.
- Allocate more budget to mobile placements.
